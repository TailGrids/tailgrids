# Field

The React Field component helps you build form fields with labels, inputs, helper text, and error messages. It keeps everything connected and structured in one place.

It makes forms easier to build and more accessible by linking labels, descriptions, and errors to the input. For example, a field can show a label, helper text, and an error message for an email input.

## Anatomy

Import the sub-components you need and compose them together.

```tsx
import {
  FieldGroup,
  Field,
  FieldLabel,
  FieldTitle,
  FieldDescription,
  FieldError,
  FieldSeparator,
  ComposedField
} from "@/components/tailgrids/core/field";

export const FieldAnatomy = () => (
  <FieldGroup>
    <Field>
      <FieldLabel htmlFor="input-id">Label</FieldLabel>
      {/* Input, Select, Textarea, etc. */}
      <FieldDescription>Optional helper text.</FieldDescription>
      <FieldError>Validation message.</FieldError>
    </Field>

    <FieldSeparator />

    <Field orientation="horizontal">
      <FieldTitle>Title</FieldTitle>
      <FieldDescription>Helper text.</FieldDescription>
      {/* Input, Select, Textarea, etc. */}
    </Field>

    <ComposedField
      label="Label"
      description="Helper text"
      errorMessage="Error message"
      invalid
    >
      {/* Input, Select, Textarea, etc. */}
    </ComposedField>
  </FieldGroup>
);
```

- `Field` wraps React Aria's `TextField` primitive. Place `FieldLabel`, form controls, `FieldDescription`, and `FieldError` inside it — ARIA linking is handled automatically via context. Supports `vertical`, `horizontal`, and `responsive` orientations.
- `FieldGroup` wraps React Aria's `Group` component to stack multiple `Field` components with consistent spacing.
- `ComposedField` composes `Field`, `FieldLabel`, `FieldDescription`, and `FieldError` into a single declarative component. Uses `TextField` context to automatically wire up `aria-labelledby` and `aria-describedby`.

## API Reference

### FieldGroup

Layout wrapper that stacks `Field` components with consistent spacing. Enables container queries for responsive orientations.

| Prop        | Type              | Default | Description                                                                                 |
| :---------- | :---------------- | :------ | :------------------------------------------------------------------------------------------ |
| `children`  | `React.ReactNode` | —       | The field controls rendered inside the group.                                               |
| `disabled`  | `boolean`         | —       | Disables the grouped controls. React Aria exposes this state on the group and its children. |
| `readOnly`  | `boolean`         | —       | Marks the grouped controls as read only.                                                    |
| `className` | `string`          | —       | Additional CSS classes applied to the container.                                            |

### Field

Core wrapper for a single field.

| Prop                 | Type                                                                                  | Default      | Description                                                                       |
| :------------------- | :------------------------------------------------------------------------------------ | :----------- | :-------------------------------------------------------------------------------- |
| `orientation`        | `'vertical' \| 'horizontal' \| 'responsive'`                                          | `'vertical'` | Layout variant applied by the component's styles.                                 |
| `invalid`            | `boolean`                                                                             | —            | Marks the field as invalid.                                                       |
| `required`           | `boolean`                                                                             | —            | Marks the field as required.                                                      |
| `disabled`           | `boolean`                                                                             | —            | Disables the field.                                                               |
| `readOnly`           | `boolean`                                                                             | —            | Makes the field read only.                                                        |
| `name`               | `string`                                                                              | —            | Name used when submitting the field value in a form.                              |
| `type`               | `string`                                                                              | `'text'`     | Input type passed to the underlying text field.                                   |
| `value`              | `string`                                                                              | —            | Controlled value for the field.                                                   |
| `defaultValue`       | `string`                                                                              | —            | Uncontrolled initial value for the field.                                         |
| `onChange`           | `(value: string) => void`                                                             | —            | Called when the value changes.                                                    |
| `description`        | `React.ReactNode`                                                                     | —            | Helper text linked to the field via ARIA.                                         |
| `errorMessage`       | `React.ReactNode \| ((validation: ValidationResult) => React.ReactNode)`              | —            | Error content associated with the field.                                          |
| `validationBehavior` | `'aria' \| 'native'`                                                                  | `'aria'`     | Controls whether validation is handled with native form validation or ARIA state. |
| `validate`           | `(value: string) => ValidationError \| true \| null \| undefined`                     | —            | Validation function used to determine whether the value is valid.                 |
| `autoFocus`          | `boolean`                                                                             | —            | Whether the field should receive focus on mount.                                  |
| `enterKeyHint`       | `'enter' \| 'done' \| 'go' \| 'next' \| 'previous' \| 'search' \| 'send'`             | —            | Hints which action label to show for the virtual keyboard enter key.              |
| `inputMode`          | `'none' \| 'text' \| 'tel' \| 'url' \| 'email' \| 'numeric' \| 'decimal' \| 'search'` | —            | Hints what kind of data the user is entering.                                     |
| `autoCorrect`        | `string`                                                                              | —            | Controls browser autocorrection behavior.                                         |
| `spellCheck`         | `string`                                                                              | —            | Controls browser spell checking behavior.                                         |
| `placeholder`        | `string`                                                                              | —            | Placeholder text shown when the field is empty.                                   |
| `minLength`          | `number`                                                                              | —            | Minimum accepted text length.                                                     |
| `maxLength`          | `number`                                                                              | —            | Maximum accepted text length.                                                     |
| `pattern`            | `string`                                                                              | —            | Regular expression pattern used for validation.                                   |
| `children`           | `React.ReactNode`                                                                     | —            | The input, textarea, or other field control rendered inside the component.        |
| `className`          | `string`                                                                              | —            | Extra classes merged with the generated field styles.                             |

### FieldLabel

Styled label element for form controls.

| Prop        | Type              | Default | Description                                    |
| :---------- | :---------------- | :------ | :--------------------------------------------- |
| `children`  | `React.ReactNode` | —       | Label content.                                 |
| `htmlFor`   | `string`          | —       | Standard `htmlFor` for standalone label usage. |
| `id`        | `string`          | —       | Optional id for the label element.             |
| `className` | `string`          | —       | Additional CSS classes.                        |

### FieldTitle

Renders a title with label styling inside `FieldContent`. Useful for non-interactive labels.

| Prop        | Type              | Default | Description                                        |
| :---------- | :---------------- | :------ | :------------------------------------------------- |
| `children`  | `React.ReactNode` | —       | Title text or elements.                            |
| `slot`      | `string`          | —       | Optional slot prop supported by React Aria `Text`. |
| `className` | `string`          | —       | Additional CSS classes.                            |

### FieldDescription

Helper text slot for providing context about a form field.

| Prop        | Type              | Default         | Description                                                            |
| :---------- | :---------------- | :-------------- | :--------------------------------------------------------------------- |
| `children`  | `React.ReactNode` | —               | Helper text content.                                                   |
| `slot`      | `"description"`   | `"description"` | The description slot used by React Aria to link the text to the field. |
| `className` | `string`          | —               | Additional CSS classes.                                                |

### FieldError

Validation message slot, styled with error color.

| Prop        | Type              | Default | Description                 |
| :---------- | :---------------- | :------ | :-------------------------- |
| `children`  | `React.ReactNode` | —       | Validation message content. |
| `className` | `string`          | —       | Additional CSS classes.     |

### FieldSeparator

Visual divider. If no children are provided, this component renders the `react-aria-components` `Separator`. If children are provided, it renders a `div` with `role="separator"` and centered content.

| Prop        | Type              | Default       | Description                                                                                                                               |
| :---------- | :---------------- | :------------ | :---------------------------------------------------------------------------------------------------------------------------------------- |
| `children`  | `React.ReactNode` | —             | Optional inline content. If present, the component renders a `div` with the children centered; otherwise it renders the Aria `Separator`. |
| `role`      | `"separator"`     | `"separator"` | Present only when children are provided and the component renders a `div`.                                                                |
| `className` | `string`          | —             | Additional CSS classes.                                                                                                                   |

### ComposedField

Convenience component that renders a `Field` with optional `FieldLabel`, `FieldDescription`, and `FieldError`.

| Prop           | Type                                                                     | Default      | Description                                                                                                         |
| :------------- | :----------------------------------------------------------------------- | :----------- | :------------------------------------------------------------------------------------------------------------------ |
| `label`        | `React.ReactNode`                                                        | —            | If provided, a `FieldLabel` is rendered with this content.                                                          |
| `description`  | `React.ReactNode`                                                        | —            | If provided, a `FieldDescription` is rendered.                                                                      |
| `errorMessage` | `React.ReactNode \| ((validation: ValidationResult) => React.ReactNode)` | —            | Error content rendered inside `FieldError` when `invalid` is truthy.                                                |
| `invalid`      | `boolean`                                                                | —            | Passed to the underlying `Field` as `invalid`; when truthy and `errorMessage` is present, `FieldError` is rendered. |
| `orientation`  | `'vertical' \| 'horizontal' \| 'responsive'`                             | `'vertical'` | Layout passed to `Field`.                                                                                           |
| `required`     | `boolean`                                                                | —            | Forwarded to `Field` (maps to `isRequired`).                                                                        |
| `disabled`     | `boolean`                                                                | —            | Forwarded to `Field` (maps to `isDisabled`).                                                                        |
| `readOnly`     | `boolean`                                                                | —            | Forwarded to `Field` (maps to `isReadOnly`).                                                                        |
| `children`     | `React.ReactNode`                                                        | **required** | The form control element(s) (Input, TextArea, Select, etc.).                                                        |
| `className`    | `string`                                                                 | —            | Additional CSS classes forwarded to `Field`.                                                                        |

## Accessibility

- **Label association:** Inside `Field`, React Aria's `TextField` context automatically links labels to inputs via `aria-labelledby`. Clicking a label focuses the input.
- **ARIA linking:** `Field` wraps `TextField`, which automatically links the label, description, and error message to the input. `ComposedField` provides the same via composition.
- **Error messages:** `FieldError` displays validation messages and connects them to the input via `aria-errormessage` so screen readers can announce them.
- **Descriptions:** `FieldDescription` provides helper text that is linked to the input via `aria-describedby`.
- **Semantic grouping:** `FieldGroup` wraps React Aria's `Group` component, providing interactive state support (`isDisabled`, `isInvalid`, `isReadOnly`) for groups of related fields.
